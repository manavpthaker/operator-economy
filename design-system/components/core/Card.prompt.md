**Card** — a document-grade surface (hairline border, quiet shadow, tight corners). Use `sheet` for the top Drafting-Blue rule that reads as a drawing sheet.

```jsx
<Card sheet kicker="Unit economics" padding="lg">
  <h3>The floor is boring — and better documented.</h3>
  <p>Freelancers run AI automation at $5–6K/mo on ~$20 of tooling.</p>
</Card>
```

`padding`: none / sm / md / lg. `kicker` renders a small-caps label. Depth is paper tone, never a heavy float.
