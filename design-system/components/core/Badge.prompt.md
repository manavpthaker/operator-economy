**Badge** — small-caps annotation label or status tag; a drafting stamp, not a rounded pill.

```jsx
<Badge tone="accent">Blueprint №004</Badge>
<Badge tone="gold">Key figure</Badge>
<Badge tone="positive" dot>Verified</Badge>
<Badge tone="neutral">Estimate</Badge>
```

Tones: `neutral` / `accent` / `gold` / `positive` / `negative`. Add `dot` for a status marker.
