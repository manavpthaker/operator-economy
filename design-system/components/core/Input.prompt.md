**Input** — text field for the site's single capture (email list). Sunken paper well, hairline border, blue focus ring.

```jsx
<label className="oe-label">Work email</label>
<Input type="email" placeholder="you@company.com" />
<Input mono placeholder="000-0000" invalid />
```

`mono` for numeric/code entry, `invalid` for the brick error border, `size`: sm / md / lg. Label it with `<label className="oe-label">`.
