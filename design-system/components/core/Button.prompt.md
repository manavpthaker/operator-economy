**Button** — the single restrained call-to-action; one primary per view, never neon or gradient.

```jsx
<Button variant="primary" size="md">Download the blueprint</Button>
<Button variant="secondary">Read the archive</Button>
<Button variant="ghost" as="a" href="#">Watch on YouTube</Button>
```

Variants: `primary` (Drafting Blue fill), `secondary` (outline), `ghost` (text). Sizes: `sm` / `md` / `lg`. Use `as="a"` for links, `fullWidth` in forms.
